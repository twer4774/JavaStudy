# 7-2. 배열 - 다차원배열

## 다차원 배열

### 이차원 배열

```java
int[][] arr = new int[2][3]; //행렬
int[][] arr = {{1,2,3}, {4,5,6}};
```

```java
public class TwoDimension {

    public static void main(String[] args) {
        int[][] arr = {{1, 2, 3}, {4, 5, 6}};

        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[i].length; j++) {
                System.out.println(arr[i][j]);
            }
            //행 출력이 끝난 후 한 줄 띄움
            System.out.println();
        }
    }
}
/*
1
2
3

4
5
6
*/
```

## 가변배열 (자바의 정석)

- 2차원 이상의 배열을 '배열의 배열'의 형태로 처리한다는 사실을 이용하면 보다 자유로운 형태의 배열을 구성할 수 있습니다.
- 다차원 배열을 생성할 때 전체 배열 차수 중 마지막 차수의 길이를 지정하지 않고, 각기 다른 길이로 설정하면 가변배열을 만들 수 있습니다.

```java
//고정 배열 생성
int[][] score = new int[5][]; //두 번째 차원의 길이는 지정하지 않습니다.
score[0] = new int[3];
score[1] = new int[3];
score[2] = new int[3];
score[3] = new int[3];
score[4] = new int[3];

//가변 배열 생성
int[][] score = new int[5][]; //두 번째 차원의 길이는 지정하지 않습니다.
score[0] = new int[4];
score[1] = new int[3];
score[2] = new int[2];
score[3] = new int[3];
score[4] = new int[3];

//중괄호를 이용한 초기화 가능
int[][] score = {
  {100, 100, 100, 100}
  ,{20, 20, 20}
  ,{30, 30}
  ,{40, 40}
  ,{50, 50, 50}
};
```
