# 7-2. 배열 - 다차원배열

## 다차원 배열

### 이차원 배열

```java
int[][] arr = new int[2][3]; //행렬
int[][] arr = {{1,2,3}, {4,5,6}};
```

```java
public class TwoDimension {

    public static void main(String[] args) {
        int[][] arr = {{1, 2, 3}, {4, 5, 6}};

        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[i].length; j++) {
                System.out.println(arr[i][j]);
            }
            //행 출력이 끝난 후 한 줄 띄움
            System.out.println();
        }
    }
}
/*
1
2
3

4
5
6
*/
```
