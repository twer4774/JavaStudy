# 13-2. 람다식

- 함수형 프로그래밍 방법

```java
(int x, int y) -> { return x+y; }
```

## 람다식 문법 살펴보기

```java
//매개변수 자료형과 괄호 생략(매개 변수(str)가 2개인 경우 생략 불가)
str -> {System.out.println(str);}

//중괄호 생략하기
str -> System.out.println(str);

//return 생략하기
(x,y) -> x+y
str -> str.length()
```

## 람다식 사용하기

```java
public interface MyNumber {
    int getMax(int num1, int num2); //추상 메소드 선언
}
//================================================
package lambda;

public class TestMyNumber {
    public static void main(String[] args) {
        //람다식을 인터페이스형 max 변수에 대입
        MyNumber max = (x, y) -> (x >= y) ? x : y;
        //인터페이스형 변수로 메소드 호출
        System.out.println(max.getMax(10, 20));
    }
}
/*
20
*/
```

## 함수형 인터페이스

- 람다식을 구현하기 위해 함수형 인터페이가 필요합니다.
  - 자바에서는 참조 변수 없이 메소드를 호출 할 수 없기 때문에 필요합니다.
  - 람다식은 익명함수로 구현하기 때문에 하나의 메소드만 선언한 인터페이스를 구현할 수 있습니다.

```java
//람다식은 익명함수로 구현하기 때문에 하나의 메소드만 선언한 인터페이스를 구현할 수 있습니다.
//=>사용하는 부분에서 어떤 메소드를 가져올지 모름(익명함수)
//==> 람다를 구현한 인터페이스에 @FunctionalInterface 추가하여 두개의 메소드를 선언하면 오류 발생
@FunctionalInterface //메소드를 하나 이상 선언하면 오류 발생
public interface MyNumber{
  int getMax(int num1, int num2);
  int add(int num1, int num2);
}
```

## 객체 지향 프로그래밍 방식과 람다식 비교

- 객체지향 방법

```java
package lambda.OOP;

public interface StringConcat {
    public void makeString(String s1, String s2);
}
//================================================
package lambda.OOP;

public class StringConcatImpl implements StringConcat{

    @Override
    public void makeString(String s1, String s2) {
        System.out.println(s1 + "," + s2);
    }
}
//================================================
package lambda.OOP;

public class TestStringConcat {
    public static void main(String[] args) {
        String s1 = "Hello";
        String s2 = "World";
        StringConcatImpl concat1 = new StringConcatImpl();
        concat1.makeString(s1, s2);
    }
}
/*
Hello,World
*/
```

- 람다식으로 구현

```java
public interface StringConcat {
    public void makeString(String s1, String s2);
}
//================================================
package lambda.Functional;

public class TestStringConcat {
    public static void main(String[] args) {
        String s1 = "Hello";
        String s2 = "World";

        StringConcat concat2 = (s, v) -> System.out.println(s + "," + v);
        concat2.makeString(s1, s2);

    }
}
/*
Hello,World
*/
```
